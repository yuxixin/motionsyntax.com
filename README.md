# MotionSyntax GitHub Pages Site

Static official website for `https://www.motionsyntax.com/`.

## Contents

- `index.html`: company homepage.
- `CNAME`: GitHub Pages custom domain, fixed to `www.motionsyntax.com`.

## GitHub Pages Setup

Recommended repository: `yuxixin/motionsyntax.com`.

Publish source:

- Branch: `main`
- Folder: `/`
- Custom domain: `www.motionsyntax.com`
- Enforce HTTPS: enabled after DNS verification succeeds.

DNS record:

```text
Type: CNAME
Host: www
Value: yuxixin.github.io
```

If the DNS provider requires a trailing dot, use:

```text
yuxixin.github.io.
```
